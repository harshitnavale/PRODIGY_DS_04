# PRODIGY_DS_04

## Task 04: Sentiment Analysis and Visualization

### Objective
Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes towards different brands and topics.

### Dataset
Twitter Sentiment Analysis Dataset

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Steps Performed
1. Loaded and explored the dataset.
2. Handled missing values.
3. Analyzed sentiment distribution.
4. Identified the most discussed brands.
5. Visualized sentiment patterns across brands.
6. Generated insights from social media sentiment data.

### Visualizations
- Sentiment Distribution
- Top 10 Most Discussed Brands
- Sentiment by Top 5 Brands
